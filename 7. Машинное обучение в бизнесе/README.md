# 7. Машинное обучение в бизнесе
## Цель проекта:
Построить модель машинного обучения для предсказания объемов запасов нефти и определения наиболее прибыльного региона для ее добычи.
## Задачи проекта: 
- предобработка данных
- обучение и проверка модели линейной регрессии
- расчет минимального объема сырья для безубыточной разработки новой скважины
- расчет прибыли и вероятности получения убытков в каждом из регионов
## Данные:
- id — уникальный идентификатор скважины
- f0, f1, f2 — три зашифрованных признака скважин 
- product — объём запасов в скважине (тыс. баррелей)
## Используемые инструменты: 
- pandas
- sklearn
- matplotlib
- seaborn
